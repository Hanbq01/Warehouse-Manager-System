# Warehouse-Manager-System
# 先看这里！
这是一个简单的仓库管理系统。是自己的课程论文项目，能力有限，该项目在功能上仍有许多不足，请见谅。

本项目使用Eclipse编写，使用Java8和MySQL8环境，如需运行，请自行适配。

数据库结构和数据在sql文件夹中，请自行导入。

启动项在src/main/start.java，本项目仅供学习参考，不可真正投入使用。
